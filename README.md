## minigrep  项目练习代码

Rust语言圣经(Rust Course)：https://course.rs/basic-practice/envs.html

项目实现

```
# 新建项目
cargo new minigrep 
# 运行项目

cargo run the poem.txt

or 

cargo build 

or rustc main.rs
```
